List In Html

A list in HTML is a way to group related items together.
Lists help organize information clearly, whether it's a shopping list, a set of instructions, or definitions.

Type of List in HTML:

There are mainly three type of lists in HTML:

  1. Ordered list.
  2. Unordered list.
  3. Definition list.

Styling List in CSS:

Lets style html list in css by using various list properties and its values:-

1. Ordered List:-

An ordered list displays items in a specific order, such as step-by-step instructions.
Items are automatically numbered or lettered.

Ordered list properties in CSS:

  1. list-style-type: Defines the numbering or bullet style.

    Values: decimal (1, 2, 3), upper-roman (I, II, III), lower-alpha (a, b, c), upper-alpha (A, B, C), lower-greek, etc.

  2. list-style-position: Controls where the numbers or bullets appear relative to the text.

    Values: inside, outside.

  3. list-style (Shorthand): Shorthand property for all list style properties.

    Values: <type> <position>.

2. Unordered List:-

An unordered list displays items in no specific order. Each item appears with a bullet symbol (●, ○, or ■) by default.

Unordered list properties in CSS:

  1. list-style-type: Defines the bullet style used in an unordered list.

    Values: disc (● - default filled circle), circle (○ - hollow circle), square (■ - square bullet), none (no bullet shown).

  2. list-style-position: Controls where the bullets appear relative to the list text.

    Values: inside (bullet appears inside the text area), outside (bullet appears outside the text area - default).

  3. list-style (Shorthand): Shorthand property for all list style properties.

    Values: <type> <position> or <type> <position> <image>.

3. Definition List:-

A definition list (<dl>) is used to display a list of terms and their definitions. Each term is defined using a pair of elements: <dt> (definition term) and <dd> (definition description).

Definition list elements in HTML:

  1. <dl> (Definition List): It acts as a container that holds all the terms and their definitions.

    Used to group together related name-value pairs.

  2. <dt> (Definition Term): Specifies the term, name, or item being defined.

    Appears in bold by default in browsers.

  3. <dd> (Definition Description): Provides the meaning, description, or value of the term specified in <dt>.

    Usually indented under the term for clarity.
