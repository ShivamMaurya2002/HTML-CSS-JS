CSS Box Model:

The box model allows us to add a border around elements, and to define space between elements.
The CSS Box Model is a way the browser represents and calculates the layout of elements.

Every HTML element is treated as a rectangular box, and this box is made up of four parts:

  • 1. Content:- The actual content of the element (text, image, etc.).
  • 2. Padding:- Space between the content and the border (transparent area).
  • 3. Border:- A line wrapping around the padding and content.
  • 4. Margin:- Space outside the border, separating this element from others.

    1. Content in Box Model:-

    This is where your actual element data (text, image, video, child elements) lives.
    It is defined by the properties:

    Properties of Content in Box Model in CSS:

  • Width:- sets the content's horizontal size.
  • Height:- sets the content's vertical size.
  • Min-width / Max-width:- restrict the smallest/largest width.
  • Min-height / Max-height:- restrict the smallest/largest height.
  • Overflow:- controls what happens if content is bigger than the box.

    • Values of Content's Properties:

  • Length:- px, em, rem, cm, etc.
  • Percentage (%):- relative to the containing block's width/height.
  • auto:- browser decides (default).
