CSS Box Model:

The box model allows us to add a border around elements, and to define space between elements.
The CSS Box Model is a way the browser represents and calculates the layout of elements.

Every HTML element is treated as a rectangular box, and this box is made up of four parts:

  • 1. Content:- The actual content of the element (text, image, etc.).
  • 2. Padding:- Space between the content and the border (transparent area).
  • 3. Border:- A line wrapping around the padding and content.
  • 4. Margin:- Space outside the border, separating this element from others.

    1. Content in Box Model:-

    This is where your actual element data (text, image, video, child elements) lives.
    It is defined by the properties:

    Properties of Content in Box Model in CSS:

  • Width:- sets the content's horizontal size.
  • Height:- sets the content's vertical size.
  • Min-width / Max-width:- restrict the smallest/largest width.
  • Min-height / Max-height:- restrict the smallest/largest height.
  • Overflow:- controls what happens if content is bigger than the box.

    • Values of Content's Properties:

  • Length:- px, em, rem, cm, etc.
  • Percentage (%):- relative to the containing block's width/height.
  • auto:- browser decides (default).

    • 2. Padding in Box Model:-

    Padding is the space between the content and the border.
    It is always transparent (inherits the element's background color).

    Properties of Padding in Box Model in CSS:

  • padding (shorthand):- Sets inner spacing (inside border). Accepts 1-4 values.
  • padding-top:- Padding on the top side.
  • padding-right:- Padding on the right side.
  • padding-bottom:- Padding on the bottom side.
  • padding-left:- Padding on the left side.

    • Note:- Values can be in px, em, rem, %, etc

  • padding:10px; → all sides 10px.
  • padding: 10px 20px; → top/bottom 10px, left/right 20px.
  • padding: 10px 20px 30px; → top 10px, left/right 20px, bottom 30px.
  • padding: 10px 20px 30px 40px; → top, right, bottom, left.

    • 3. Border in Box Model:-

    The border wraps around padding + content.
    It can be visible (colored, styled) or invisible (none).

    Properties of Border in Box Model in CSS:

  • border (shorthand):-
  • border-width:- sets thickness of the border.
  • border-style:- solid, dashed, dotted, double, groove, ridge, inset, outset, none.
  • border-color:- sets color of the border.
  • border-top:-A shorthand property to set border-width, border-style, and border-color only for the top side.
  • border-right:- A shorthand property to set border-width, border-style, and border-color only for the right side.
  • border-bottom:- A shorthand property to set border-width, border-style, and border-color only for the bottom side.
  • border-left:- A shorthand property to set border-width, border-style, and border-color only for the left side.
  • border-radius:- Rounds the corners of the border. Values in px, %, em.
  • border-collapse:- (used in tables) Defines whether table borders are collapsed into a single border or separated.
  • border-spacing:- (used in tables) Specifies the distance between the borders of adjacent table cells (works only when border-collapse is separate).

    • Values of Border's Properties:

  • Length:- px, em, rem, etc.
  • Color values:- named colors, hex (#000), rgb(), hsl(), transparent.
  • styles:- solid | dashed | dotted | double | groove | ridge | inset | outset | none

    • 4. Margin in Box Model:-

    Margin is the outermost space around the element's border.
    It separates elements from each other.
    Unlike padding, margin is transparent and does not take background color.

    Properties of Margin in Box Model in CSS:

  • margin (shorthand):- Sets outer spacing (outside border). Accepts 1-4 values.
  • margin-top:- Margin on the top side.
  • margin-right:- Margin on the right side.
  • margin-bottom:- Margin on the bottom side.
  • margin-left:- Margin on the left side.

    • Note:- Margin values can be in px, em, rem, %, auto, etc.

  • margin: 20px; → all sides 20px
  • margin: 10px 20px; → top/bottom 10px, left/right 20px
  • margin: 10px 20px 30px; → top 10px, left/right 20px, bottom 30px
  • margin: 10px 20px 30px 40px; → top, right, bottom, left
  • margin: auto; → used for horizontal centering
