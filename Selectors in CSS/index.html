CSS Selector

CSS selectors are used to find or select the HTML elements we want to style. (or)
CSS selectors are patterns used to select and style HTML elements. They tell the browser which element(s) the CSS rules should apply to.

Types of Selectors in CSS:

CSS selectors are mainly divided in five categories, they are:-

  • Simple Selectors: Select elements by their name, id, or class.
  • Combinator Selectors: Select elements based on their relationship with other elements (like inside, next to, etc).
  • Pseudo-class Selectors: Select elements in a special state (like hover, first child, etc).
  • Pseudo-element Selectors: Select and style part of an element (like first letter, first line, etc).
  • Attribute Selectors: Select elements using their attributes or attribute values (like [type="text"]).

1. Simple Selectors:

Simple selectors are the most basic CSS selectors that are used to select HTML elements directly without considering their relationship with other elements.
They target elements based on their name, id, class, or attributes.

    • 1. ID Selectors:-

    The ID selector in CSS is used to style a specific element by targeting its unique id attribute.
    Since an ID must be unique within a page, it is used to select only one element.
    To use an ID selector, write a hash symbol (#) followed by the ID name.

    Example: #header { color: blue; }

    2. Class Selector:-

    The class selector uses the class attribute of HTML elements to select one or multiple elements.
    Unlike IDs, classes are not unique and can be applied to many elements on the same page.
    To select elements with a specific class, write a dot (.) character, followed by the class name.

    Example: .para { color:black; }

    3. Tag Name Selector:-

    The tag name selector (also called the type selector) is used to select all elements of a given HTML tag.
    It applies styles to every occurrence of that tag in the document.
    To use it, simply write the name of the HTML tag without any symbol.

    Example: p { color: black; }
