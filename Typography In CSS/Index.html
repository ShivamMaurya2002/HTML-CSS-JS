Typography in CSS

Typography in CSS is the way we design and style text on a webpage using CSS properties like font, size, color, spacing, and alignment to make the text readable and visually appealing.
Good typography makes a website more attractive, clear, and user-friendly.

What comes under Typography in CSS?

Typography covers all the text-related CSS properties that control how text looks and how readable it is on a webpage.
They are:-

  1. Font Properties.
  2. Text Properties.
  3. Color & Background.
  4. Web Fonts & Custom Fonts.
  5. Responsive Typography.

1. Font Properties:-

Font properties in CSS are used to control the style of the text, such as the typeface (family), size, weight (boldness), and whether it appears italic or normal.
Example: Choosing Arial as the font, setting size to 16px, and making it bold.

Some Font Properties are:-

  • font-family:- Decides which font to use (like Arial, Poppins, Times New Roman).
  • font-size:- Defines the size of the text. Can be set using absolute units (px, pt) or relative units (em, rem, %, vw, vh).
  • font-weight:- Determines the thickness or boldness of the text. Values can be keywords (normal, bold, lighter, bolder) or numeric weights (100-900).
  • font-style:- Defines the style of the font. Common values: normal, italic, oblique.
  • font-variant:- Controls small-caps formatting. It displays lowercase letters as smaller uppercase letters.
  • font (Shorthand):- A shorthand property that sets multiple font properties (font-style, font-variant, font-weight, font-size/line-height, and font-family) in a single line.

    • 2. Text Properties:-

    Text properties define how text is displayed on the page, including alignment, spacing, decoration (underline/line-through), transformation (uppercase/lowercase), and shadows.
    Example: Centering a heading, making it uppercase, and adding letter spacing.

    Some Text Properties are:-

  • text-align:- Sets the alignment of text (left, right, center, justify).
  • text-decoration:- Adds or removes decorations like underline, overline, line-through.
  • text-transform:- Changes the case of text (uppercase, lowercase, capitalize).
  • line-height:- Sets space between lines of text (single, double spacing).
  • letter-spacing:- Adjusts space between letters.
  • word-spacing:- Adjusts space between words.
  • text-indent:- Adds space at the beginning of the first line of text.
  • text-shadow:- Controls how spaces and line breaks are handled (normal, nowrap, pre).

    • 3. Color & Background:-

    These properties control the text color and allow you to apply background effects to text, including gradients and advanced designs.
    Example: Making paragraph text dark gray, or creating gradient-colored headings.

  • color:- Sets the color of the text.
  • background-color:- Sets the background color of an element.
  • background-image:- Adds an image as the background of an element.
  • background-repeat:- Controls if and how the background image is repeated (repeat, no-repeat, repeat-x, repeat-y).
  • background-position:- Sets the starting position of the background image (top, center, bottom, left, right).
  • background-size:- Defines the size of the background image (cover, contain, auto, px, %).
  • background (shorthand):- A shortcut to set all background properties in one line.

    • 4. Web Fonts & Custom Fonts:-

    Web fonts allow you to use fonts beyond the default system fonts by importing them from sources like Google Fonts or using @font-face for custom fonts.
    Example: Using "Poppins" or "Roboto" font from Google Fonts for better design.

  • @font-face:- Allows you to load and use custom fonts in your website by defining the font source.
  • font-family with web fonts:- You can apply Google Fonts or any external fonts using the font-family property.
  • Google Fonts:- Free online font library provided by Google that can be imported using a link or CSS.
  • Custom Fonts:- Fonts downloaded or hosted locally and applied using @font-face.
  • Fallback Fonts:- A list of fonts given in font-family; if the first one is not available, the next is used.

    • 5. Responsive Typography:-

    Responsive typography ensures that text scales and adjusts according to the screen size or device, making it easier to read on mobiles, tablets, and desktops.
    Example: Using vw or rem units so text automatically resizes on smaller screens.

  • Relative Units:- Use units like em, rem, %, vw, and vh instead of fixed px for better scaling on different devices.
  • Media Queries:- Apply different font sizes or styles for different screen sizes using CSS @media rules.
  • Clamp():- A CSS function that sets text size with a minimum, preferred, and maximum value for responsiveness.
  • Viewport Units:- Font size adjusts based on the width or height of the screen (vw = viewport width, vh = viewport height).
  • Fluid Typography:- Technique where font sizes scale smoothly across devices using relative units and CSS functions.
